# uDrive premium mobile update

## Included

- Complete unified Flutter Customer/Driver application
- Customer mode with switch to Driver mode
- Premium left drawer and bottom navigation
- Native and animated splash screens
- Dummy mobile login and OTP (`1234`)
- Persistent login, language and selected mode
- English and Urdu RTL support
- Active customer ride-request and fare-offer simulation
- Active driver accept/counteroffer simulation
- Tourist destinations and driver-created packages
- Trips, wallet, notifications, safety and support
- Driver earnings, wallet, payout and active-trip simulation
- Four-step vehicle registration
- Multiple vehicle list and verification states
- Android, iOS and web platform scaffolding
- Original uDrive launcher icon and splash assets
- Railway Dockerfile
- Root GitHub Actions workflow for APK and AAB

## Important

The icon is an original uDrive route/U mark. It follows the friendly visual direction requested but is not an exact copy of the inDrive trademark.
