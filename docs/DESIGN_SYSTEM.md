# UDrive Design System

## Brand colors

- Primary emerald: `#0F766E`
- Travel blue: `#2563EB`
- Kashmir gold: `#F59E0B`
- Canvas: `#F6F8FB`
- Main text: `#0F172A`
- Secondary text: `#64748B`
- Danger: `#DC2626`

## Brand mark

The starter uses an original rounded route symbol with an emerald-to-blue gradient. It intentionally avoids copying another company's trademarked logo.

## UI principles

- Large touch targets for drivers using the app while parked
- Strong price visibility during offer negotiation
- Clear distinction between ride offers and tour packages
- Safety actions remain visually prominent
- Urdu uses native RTL direction throughout the application
- Cards use 18–24 px corner radii with light borders instead of heavy shadows

## Core statuses

### Rides
`requested`, `offers_received`, `driver_selected`, `driver_arriving`, `in_progress`, `completed`, `cancelled`

### Driver packages
`draft`, `pending_approval`, `changes_required`, `approved`, `active`, `paused`, `rejected`, `suspended`, `expired`
